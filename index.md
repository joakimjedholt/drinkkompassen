# Privacy Policy för Drinkkompassen

**Senast uppdaterad:** 30 januari 2026

## Översikt

Drinkkompassen ("appen", "vi") respekterar din integritet. Denna policy förklarar vilken data vi samlar in och hur den används.

## Data vi samlar in

### Via CloudKit (Apple)
Appen använder Apple CloudKit för att synkronisera din data mellan dina enheter:
- **Ditt personliga barskåp** - flaskor du har lagt till
- **Dina egna drinkrecept** - recept du skapat
- **Dina favoriter** - drinkar du favoritmarkerat
- **Inställningar** - temaval, sortering, preferenser

### Lokalt på din enhet
- **Receptcache** - för offline-åtkomst
- **Senaste synktid**

### AI-receptskanning
När du använder AI-skanningsfunktionen:
- **Bilder** - skickas tillfälligt till Claude API (Anthropic) för OCR och textanalys
- **Text** - extraherad recepttext processas av AI
- **Bilder och text raderas direkt** efter processning
- **Ingen permanent lagring** hos tredje part

## Hur vi använder data

- **Synkronisering** - CloudKit synkar din data mellan dina Apple-enheter
- **Rekommendationer** - Dagens rekommendation baseras på ditt barskåp och säsong
- **Matchning** - Smart matchning mellan ditt barskåp och recept
- **Offline-tillgång** - Cachad data låter dig använda appen utan internet

## Delning med tredje part

### Community-funktionen
Om du väljer att dela ett recept med community:
- **Ditt recept blir publikt synligt** för alla användare
- **Ingen personlig information** (namn, email, etc) delas
- **Receptet granskas** innan publicering

### Vi delar INTE din data med:
- Reklamföretag
- Databrokers
- Andra tredje parter för marknadsföring

## Datasäkerhet

- **CloudKit** - All molndata lagras säkert via Apples CloudKit
- **Krypterad överföring** - Data skickas krypterat
- **Apple ID** - Endast du kan komma åt din CloudKit-data via ditt Apple ID

## Dina rättigheter

Du kan när som helst:
- **Radera din data** - via appen eller iCloud-inställningar
- **Stänga av CloudKit** - appen fungerar lokalt utan synk
- **Exportera din data** - kontakta oss för dataexport

## Tredjepartstjänster

### Claude API (Anthropic)
- Används för AI-receptskanning
- Bilder och text processas tillfälligt
- Ingen permanent lagring av ditt innehåll
- Läs mer: https://www.anthropic.com/privacy

### Apple CloudKit
- Används för molnsynkronisering
- Läs mer: https://www.apple.com/legal/privacy/

## Barn

Appen är klassad 17+ på grund av alkoholrelaterat innehåll. Vi samlar inte medvetet in data från personer under 17 år.

## Ändringar i policyn

Vi kan uppdatera denna policy från tid till annan. Den senaste versionen finns alltid på denna sida. Stora ändringar meddelas i appen.
